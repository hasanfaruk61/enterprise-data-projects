# Enterprise Data Projects

This repository contains structured data engineering projects built progressively from fundamentals to advanced concepts.

The goal of this repository is to demonstrate:

- Strong Python fundamentals
- NumPy-based numerical thinking
- Pandas-based data manipulation
- Data cleaning and transformation workflows
- Modular project architecture
- Decision-support analytics mindset
